Simulador de Uber
=================

Bienvenido a la documentación del Simulador de Uber de Teoria-1800
---------------------------------------------------

Este proyecto se trata sobre la creacion de una web app que de una
          idea sobre lo que uno como Uber driver necesita para ser fructifero en
          su emprendimiento. Cientas de personas en Honduras dependen de menos
          de salario minimo. Y miran su vehiculo como una herramienta para poder
          generar algo de dinero de forma accesible. Pero es dificil considerar
          que tan caro puede ser el uber a largo plazo o siquiera si sera algo
          rentable a futuro. De ahi nace la idea de un simulador de dia de uber.

Introducción
------------

El Simulador de Uber está diseñado para replicar la lógica y las operaciones básicas de un servicio de transporte bajo demanda.

Características principales:

- **Simulación de viajes:** Configura y ejecuta simulaciones de viajes desde un punto A a un punto B.
- **Precios dinámicos:** Implementa un sistema de precios dinámico basado en trafico.
- **Personalización:** Modifica parámetros como el el consumo de combustible, etc.
